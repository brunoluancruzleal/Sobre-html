O que é HTML

Linguagem de marcação

Sobre a Linguagem:

A estrutura HTML consiste em elementos organizados hierarquicamente, começando com a declaração do tipo de documento para identificar o HTML. O elemento principal é o HTML, que contém dois grandes blocos: o cabeçalho e o corpo. O cabeçalho guarda informações sobre a página, como metadados, título e links externos para estilos e scripts. Ele não contém conteúdo visível diretamente na página. O corpo é onde fica todo o conteúdo visível, como textos, imagens, vídeos e formulários. Elementos como parágrafos, cabeçalhos, listas e tabelas são usados para organizar e apresentar as informações. Com o HTML5, surgiram tags semânticas, como cabeçalho, navegação, seção e rodapé, que ajudam a estruturar o conteúdo de forma mais clara e acessível. Essas tags tornam o código mais legível e melhoram a interpretação por navegadores e mecanismos de busca.

Historia da Linguagem

O HTML (HyperText Markup Language) foi criado por Tim Berners-Lee em 1989 no CERN como uma forma de compartilhar documentos interligados por hiperlinks. Lançado oficialmente em 1991, com apenas 18 tags, evoluiu rapidamente com a padronização pelo IETF e, posteriormente, pelo W3C. O HTML 2.0, em 1995, consolidou seu uso inicial, enquanto o HTML 3.0 e 4.0 trouxeram avanços como suporte a tabelas, formulários e estilos com CSS. Durante a década de 2000, o XHTML tentou tornar a linguagem mais rigorosa, mas foi superado pelo HTML5, lançado em 2014, que modernizou a web com suporte a multimídia, interatividade e dispositivos móveis. Hoje, o HTML segue um modelo de padrão vivo, evoluindo continuamente para atender às necessidades da web moderna.

2. Elementos Principais

  • DOCTYPE html:
    • Informa ao navegador que este é um documento HTML5.
    • Deve ser sempre a primeira linha do código.
  • HTML:
    • É o elemento raiz de um documento HTML.
    • Todo o conteúdo da página deve estar dentro dessa tag.
    • O atributo lang indica o idioma principal do documento (exemplo: lang="pt-BR" para português brasileiro).
  • Head:
    • Contém metadados e informações sobre o documento.
    • Não exibe conteúdo diretamente na página.

      • inclui:

    • meta: Define informações como o conjunto de caracteres e viewport.
    • title: Define o título exibido na aba do navegador.
    • Links para arquivos externos como CSS e scripts (link e script).
  • body:
    • Contém todo o conteúdo visível da página.
    • Inclui textos, imagens, vídeos, tabelas, links, formulários, etc.
